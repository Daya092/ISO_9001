PLANTILLAS DE DOCUMENTOS ISO 9001
====================================

Esta carpeta contiene las plantillas de documentos que los usuarios pueden descargar.

ARCHIVOS REQUERIDOS:
- manual_calidad.xlsx
- politica_calidad.xlsx
- procedimientos.xlsx
- registros_calidad.xlsx
- plan_capacitacion.xlsx

INSTRUCCIONES:
1. Crear archivos Excel con las plantillas correspondientes
2. Nombrar los archivos exactamente como se indica arriba
3. Colocar los archivos en esta carpeta
4. Los usuarios podrán descargar estas plantillas desde el módulo de Capacitación

NOTA: Los archivos deben estar en formato Excel (.xlsx) para compatibilidad.
